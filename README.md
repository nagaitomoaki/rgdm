# Getting Started
## Prerequisites
* Android Studio 4.0.

## Set up environment with docker
    sudo docker build -t rgdm
    
## Installation


    git clone https://github.com/nagaitomoaki/rmgd.git
    cd rmgd
    jupyter notebook restart.ipynb
